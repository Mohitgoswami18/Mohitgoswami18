## 🚀 What I'm Building

### [Vyom](https://github.com/Mohitgoswami18/Vyom) — AI Job Application Agent

> Applying to internships is repetitive, time-consuming, and broken. Vyom fixes that.

Vyom is an AI-powered agent that reads job listings, fills applications, and personalises cover letters — so you spend time preparing for interviews, not copy-pasting your resume for the 40th time.

**What makes it interesting:**
- Agent architecture that parses job descriptions and matches them against a user's profile
- Auth via Clerk, REST APIs, and a MongoDB-backed user data model
- React + Tailwind frontend with a Node/Express + FastAPI dual backend
